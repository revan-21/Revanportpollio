# 🌐 Revan Portfolio

A personal portfolio website designed to showcase skills, projects, education, and contact information. This portfolio highlights frontend development abilities with a modern, responsive, and user-friendly interface.

## 🚀 Features

* Responsive modern design
* Smooth navigation between sections
* About Me section
* Skills showcase
* Projects gallery
* Contact form and social links
* Interactive UI with animations

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Sections Included

* Home
* About
* Skills
* Projects
* Contact

## 📦 Installation

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser

## 📌 Future Improvements

* Add dark mode
* Integrate backend for contact form
* Add project filtering option
* Improve animations and UI effects

## 🌟 Purpose

This project is created to represent personal achievements, technical skills, and frontend development projects in a professional way.
